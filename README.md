# 存證信函產生器

這是一個本機網頁工具，用來填寫存證信函資料並以瀏覽器列印功能另存 PDF。

## 使用方式

1. 開啟 `index.html`，或使用本機網址 `http://127.0.0.1:8765/`。
2. 填入寄件人、收件人、副本收件人與信函內容。
3. 需要多位收件人時，按「增加收件人」。
4. 按「產出 PDF」。
5. 在列印視窗選擇「另存為 PDF」，紙張選 A4。

## 格式依據

- 每頁 10 行，每行 20 字，共 200 字。
- 英文、數字與符號可自動轉為全形。
- 版面依提供的郵局範本設定 A4 與左右約 31.7mm 邊界。
- 內容文字依使用說明採 18pt 字級、固定格線排版。
